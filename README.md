# Deploy Visio Professional 2024 LTSC
To Install Visio Professional 2024: (Windows only)

1. Clone / Download this repository using:
```sh
  git clone https://github.com/anantkaul/VisioPro2024
```
2. Copy / Move this folder `VisioPro2024` to `C:`
3. Run CMD as administrator and run the following commands to install:
```sh
	cd C:\VisioPro2024
	.\setup.exe /configure .\Configuration.xml
```
4. Wait for the setup to install. It may take a few minutes to install.
5. Enjoy your free Visio Professional 2024
